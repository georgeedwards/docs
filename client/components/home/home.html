Nativescript

Open-Source platform to build high performance cross-platform native mobile apps with web technologies. This includes tight Angular2 integration.

Get Started Learn More

Create awesome native mobile apps with JavaScript (or TypeScript).

Super Performance

Create apps with 100% native performance and power. Buttery smooth, even on Android! It's also quick to learn with an awesome community, plenty of resources and high performance workflows.

Native and Extensible

Access all native iOS, Android and Windows APIs directly with JavaScript. Easily reuse existing plugins from npm, CocoaPods (iOS) and Gradle (Android) directly in NativeScript projects, plus hundreds of NativeScript specific plugins on npm

Web Technologies

Work with JavaScript or TypeScript, HTML and CSS (or SASS and LESS). Close links to the Angular team and full support for Angular 2.
